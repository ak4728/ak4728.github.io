# ak4728.github.io

A small collection of interactive mapping and visualization projects showcasing transit, bike/ped, and spatial analysis work. Key items include H3 taxi-hex visualizations, RTD bus pings (Kepler + PowerBI), Denver bike & pedestrian maps, traffic analyses, and demonstration pages.

Quick links (Medium articles):

- H3 taxi pickups (Uber H3 overview & examples): https://medium.com/better-programming/playing-with-ubers-hexagonal-hierarchical-spatial-index-h3-ed8d5cd7739d
- Denver bike & ped data with Kepler.gl: https://medium.com/@abdullahkurkcu/downloading-and-visualizing-denver-bike-and-ped-data-with-python-and-kepler-gl-f0b030b7eac5

Repository notes:

- The original Folium/Leaflet index map is preserved as `h3_taxi_pickups.html`.
- Combined RTD bus pings (Kepler + PowerBI) are in `rtd_bus_pings.html`.
- The Denver bike map is `denver_bike_kepler.html`.
- The conceptual Morrison road-diet before/after slider is `morrison_road_diet_before_after.html`.
- The contagion SIR simulator is `contagion_simulator.html`.
